# 📚 CleanBook (교보문고/알라딘/Yes24 출판사 필터)

**CleanBook**은 온라인 서점(교보문고, 알라딘, **Yes24**)에서 원치 않는 출판사나 저자의 책을 검색 결과와 베스트셀러 리스트에서 시각적으로 차단해주는 Chrome 확장 프로그램입니다.

## 📌 주요 기능

* **국내 3대 서점 완벽 지원:**
    * **교보문고:** 검색 결과, 신간, 베스트셀러 등 전 페이지 지원
    * **알라딘:** `PublisherSearch` 링크 정밀 추적으로 오차단 방지
    * **[NEW] Yes24:** 리스트형, 타일형, 테이블형(베스트셀러) 등 다양한 레이아웃 완벽 대응
* **정밀 차단 시스템:**
    * 단순 텍스트 매칭이 아닌 DOM 구조를 분석하여 책 제목이 아닌 **저자/출판사** 정보만 정확히 필터링합니다.
    * 동명의 책 제목이 차단되는 문제를 방지했습니다.
* **편리한 관리:**
    * **검색 & 정렬:** 등록된 차단 키워드가 많아도 검색 기능과 가나다순 정렬로 쉽게 관리할 수 있습니다.
    * **하이라이팅:** 내가 입력한 검색어가 목록에서 어디에 있는지 색상으로 강조됩니다.
* **데이터 백업/복구:**
    * 차단 목록을 JSON 파일로 안전하게 백업하고, 다른 컴퓨터에서 복구할 수 있습니다.

## 🛠 설치 방법 (개발자 모드)

1.  이 저장소를 `git clone` 하거나 ZIP 파일로 다운로드하여 압축을 풉니다.
2.  Chrome 브라우저 주소창에 `chrome://extensions`를 입력합니다.
3.  우측 상단의 **'개발자 모드'**를 켭니다.
4.  **'압축해제된 확장 프로그램을 로드합니다'** 버튼을 클릭하고 프로젝트 폴더를 선택합니다.

## 📝 사용 방법

1.  브라우저 우측 상단의 **CleanBook 아이콘**을 클릭합니다.
2.  차단하고 싶은 **출판사명** 또는 **저자명**을 입력하고 '추가'를 누릅니다.
3.  서점 사이트를 새로고침하면 해당 출판사/저자의 책이 가림막으로 차단됩니다.
4.  **백업/복구** 버튼을 통해 차단 목록을 파일로 저장해둘 수 있습니다.
